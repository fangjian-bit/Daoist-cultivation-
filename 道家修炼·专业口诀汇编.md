# 道家修炼·专业口诀汇编

---

## 前言

本文档是《修炼原理与口诀》的深化扩展篇，汇编了道家修炼中更为专业的口诀、咒语、存思法门。

包括：打坐前请护法、净坛结界、金光护身、存思观火、内丹密诀等。这些内容源自正统道藏和师传口诀，是真正"关起门来练"的东西。

> **说明**：本文档作为传统文化知识整理。部分内容涉及道教仪轨，日常修炼以清净自然为主，不必强求全套仪式。可根据个人情况选用。

---

## 一、修炼前准备——八大神咒

八大神咒是道教日常修持的基础，打坐、诵经、画符前皆可使用。初学不必全背，选前三咒即可。

### 1.1 净心神咒——打坐必诵

> 这是最重要的入门口诀，意在清净心神，去除杂念。

```
太上台星，应变无停。
驱邪缚魅，保命护身。
智慧明净，心神安宁。
三魂永久，魄无丧倾。
```

**用法**：打坐前，闭目默诵三遍。诵时意想心中杂念如乌云散去，一轮明月从胸中升起。

**白话释义**：祈请大道之力，驱除外邪、保住心神，让智慧清明，三魂七魄安稳不乱。

---

### 1.2 净口神咒——养气必诵

> 用于修炼前清净口业，尤其适合话多耗气后。

```
丹朱口神，吐秽除氛。
舌神正伦，通命养神。
罗千齿神，却邪卫真。
喉神虎贲，炁神引津。
心神丹元，令我通真。
思神炼液，道炁常存。
```

**用法**：打坐前默诵一遍，诵后以舌搅口腔，待津液满口后分三口咽下，意念送入丹田。

**核心要点**：叩齿 + 咽津。默诵时配合叩齿（上下牙轻轻相叩）36 下，刺激唾液分泌。道家称唾液为"金津玉液""长生酒"。

---

### 1.3 净身神咒——驱浊净体

> 用于去除身体浊气，清净气场。

```
灵宝天尊，安慰身形。
弟子魂魄，五脏玄冥。
青龙白虎，队仗纷纭。
朱雀玄武，侍卫我身。
```

**用法**：打坐前默诵一遍，诵时意想四灵神兽（青龙在左、白虎在右、朱雀在前、玄武在后）守护自身。

---

### 1.4 安土地神咒——祭告地祇

> 用于告知土地神、安定修炼场所的气场。

```
元始安镇，普告万灵。
岳渎真官，土地祇灵。
左社右稷，不得妄惊。
回向正道，内外澄清。
各安方位，备守坛庭。
太上有命，搜捕邪精。
护法神王，保卫诵经。
皈依大道，元亨利贞。
```

**用法**：在自己固定修炼场所第一次使用时诵三遍，之后每次修炼诵一遍即可，告明此地为修炼之所。

---

### 1.5 金光神咒——护身结界

> **核心口诀之一**。金光咒是道家护身法的基础，用途极广。

```
天地玄宗，万炁本根。
广修万劫，证吾神通。
三界内外，惟道独尊。
体有金光，覆映吾身。
视之不见，听之不闻。
包罗天地，养育群生。
诵持万遍，身有光明。
三界侍卫，五帝司迎。
万神朝礼，役使雷霆。
鬼妖丧胆，精怪亡形。
内有霹雳，雷神隐名。
洞慧交彻，五炁腾腾。
金光速现，覆护真人。
```

**用法**：

**日常护身**：清晨面向东方，默诵一遍。诵时观想（想象）体内发出金色光芒，从丹田扩散到全身，然后扩展到整个房间。

**打坐前结界**：在修炼座位上面向东方站立，诵金光咒一遍。诵到最后一句"覆护真人"时，观想一个金色光罩从头顶落下，将自己完全罩住。

**遇事心慌时**：立即默念"体有金光，覆映吾身"一句，配合深呼吸。

---

### 1.6 净天地神咒——清理修炼场所

```
天地自然，秽炁分散。

八方威神，使我自然。
灵宝符命，普告九天。
乾罗答那，洞罡太玄。
斩妖缚邪，度人万千。
中山神咒，元始玉文。
持诵一遍，却病延年。
按行五岳，八海知闻。
魔王束首，侍卫我轩。
凶秽消散，道炁长存。
```

**用法**：搬家、入住新场所、或感觉房间气场不好时，在房间中央站立，诵一遍。诵时意想一道白光从头顶灌入，扩散冲洗整个空间。

## 二、打坐护法口诀

### 2.1 入座请护法

> 在正式打坐前，可默念请护法口诀。护法不是外在鬼神，而是自身正气所感召的守护力量。

**简单版（推荐日常）**：

```
弟子（或"某"）今于此地静坐修炼，
祈请天地正气，护佑坛场。
左有青龙，右有白虎，
前有朱雀，后有玄武。
四大神兽，各安方位。
邪魔外道，不得侵扰。
吾今入静，万法不沾。
急急如律令。
```

**用法**：面朝东方或南方，双手结太极印（两手拇指掐中指，余指相抱）置于胸前，默念一遍。然后缓缓放下双手，开始静坐。

### 2.2 护法二十四将召请（专业版）

> 道教雷法中二十四护法天将的召请口诀，此处仅作知识介绍。

```
仰启玄天大圣者，北方壬癸至灵神。
金阙真尊应化身，无上将军号真武。
威容赫赫在乾坤，左右天兵守吾身。
丹天伏魔大圣者，凌霄御前统万神。
……
```

> 此为专业法脉传承内容，完整口诀须有师授。日常修炼用简单版即可。

### 2.3 出定送护法

打坐结束后，应感谢护法，送归各位：

**简单版**：

```
谢过四方护法神，弟子收功。
各归本位，各安其所。
来时接引，去时送归。
太上慈悲，福生无量。
```

### 2.4 睡功护身诀

> 睡前使用，保一夜安稳。

```
夜半三更，万籁俱寂。
吾今卧眠，请神护之。
头顶华盖，足踏祥云。
左辅右弼，护我元真。
邪不敢近，梦不敢侵。
天明即起，神清气清。
急急如律令。
```

**用法**：躺下后，闭目默念一遍。配合"狮子卧"姿势（右侧卧，右手贴右脸颊）。

## 三、存思观想法门

存思，又称"存想""观想"，是道家修炼的核心技法之一。原理是以意念凝聚成形象，通过形象调动内在的气机。

### 3.1 观火法

> 观火是存思中最常用、最安全的一种。

**基础观火——丹田温火**：

1. 静坐入定后
2. 意守丹田（肚脐下三指处）
3. 在丹田中"观想"（想象）有一个乒乓球大小的火球
4. 火是温热的，不是灼烧的——如同冬日暖炉，橘红色，柔和
5. 随着吸气，火球微微发亮
6. 随着呼气，火球微微变暗（不强求，自然即可）
7. 保持 10-15 分钟

**口诀**：

```
丹田一窍，内有真火。
恍恍惚惚，温温煦煦。
不急不燥，不冷不灭。
久照长明，精自化炁。
```

**进阶观火——三昧真火观**：

在丹田温火纯熟后，可练习三火合一：

1. 丹田火（下）：橘红色，如烛光
2. 膻中火（中）：金黄色，如灯火
3. 眉心火（上）：银白色，如星芒
4. 先分别观三个火
5. 然后观想三火同时亮起
6. 最后观想三火合成一团紫金色的光

> **口诀**：**"三昧真火，煅烧凡胎。炼尽阴滓，纯阳自来。"**

> **警告**：观火用意要轻。如果出现口干舌燥、心烦上火、面红发热等症状，说明用意太重了，应减少观火时间或改为观水法。

---

### 3.2 观水法

> 观水与观火相反，用于清热降火、滋阴润燥，特别适合上火体质。

**基础观水——太一真水**：

1. 静坐入定
2. 观想丹田中不是火，而是一汪清水
3. 水是清凉的，不是冰冷的——如同山间清泉
4. 水面如镜，微微荡漾
5. 随着呼吸，水波轻轻起伏
6. 保持 10-15 分钟

**口诀**：

```
肾水为根，上朝于心。
清凉透彻，洗尽烦襟。
心火下降，肾水上升。
水火既济，道体自平。
```

---

### 3.3 观日月法——服食日月精华

> 《黄庭经》云："吞日吸月，养我灵根。" 这是上清派的核心存思法。

**观日法（清晨用，补充阳气）**：

1. 卯时（5:00-7:00），面向东方
2. 站桩或端坐
3. 闭目，观想东方天边有一轮红日（不是用眼看，是脑海中的意象）
4. 日光照耀在自己身上
5. 吸气时：观想太阳的光芒从百会穴灌入，沿中脉（身体正中线）下降至丹田
6. 呼气时：观想光芒在丹田中稳定，如一颗小太阳
7. 做 9-18 次呼吸

**口诀**：

```
东望扶桑，日之初升。
赤气煌煌，覆我身形。
取彼精华，纳我灵根。
丹元充满，光照乾坤。
```

**观月法（夜晚用，滋阴养神）**：

1. 亥时（21:00-23:00），面向西方（如果有真月亮最好，没有就在室内观想）
2. 闭目，观想头顶上方有一轮圆月，银白清凉
3. 月华如水，从头顶淋下，冲洗周身
4. 每呼吸一次，月华从头淋到脚一次
5. 做 9-18 次呼吸

**口诀**：

```
月出皎皎，光满太虚。
采其精华，以溉灵株。
水精入骨，月魄安神。
澡雪精神，浑然天真。
```

---

### 3.4 存神法——五脏神观

> 道家认为五脏各有其"神"（主管该脏腑的高级能量体），存思五脏神是高级修炼。

**五脏神对应**：

| 脏腑 | 神名 | 颜色 | 方位 | 意守位置 |
|------|------|------|------|----------|
| 肝 | 龙烟（青神） | 青 | 东 | 右肋下方 |
| 心 | 丹元（赤神） | 赤 | 南 | 胸口正中 |
| 脾 | 常在（黄神） | 黄 | 中 | 胃脘 |
| 肺 | 皓华（白神） | 白 | 西 | 右胸 |
| 肾 | 玄冥（黑神） | 紫/黑 | 北 | 后腰两侧 |

**修炼口诀**：

```
心神丹元字守灵，肺神皓华字虚成。
肝神龙烟字含明，肾神玄冥字育婴。
脾神常在字魂停，胆神龙曜字威明。
六腑五脏神体精，皆在心内运天经。
昼夜存之自长生。
```

——出自《黄庭内景经》

**修炼方法（以存思心神为例）**：

1. 静坐入定
2. 将意念收入胸口正中（膻中稍上）
3. 观想那里有一个穿红色衣服的小人（约一寸大小），端坐
4. 随着呼吸，小人微微发亮
5. 保持 5-10 分钟后，缓缓回到丹田

> **推荐顺序**：先存思心神（心主神明，最易感知），再脾神（脾主意），再肾神。

## 四、内丹修炼专业口诀

### 4.1 吕祖百字碑

> 吕洞宾所作，内丹修炼的浓缩版教科书。全篇 100 字，字字珠玑。

```
养气忘言守，降心为不为。
动静知宗祖，无事更寻谁。
真常须应物，应物要不迷。
不迷性自住，性住气自回。
气回丹自结，壶中配坎离。
阴阳生反复，普化一声雷。
白云朝顶上，甘露洒须弥。
自饮长生酒，逍遥谁得知。
坐听无弦曲，明通造化机。
都来二十句，端的上天梯。
```

**逐句白话解释**：

| 原文 | 白话 |
|------|------|
| 养气忘言守 | 养气的方法就是忘掉言语、放下执着 |
| 降心为不为 | 降伏心念的方法是"为无为"——看似在做，实际没用力 |
| 动静知宗祖 | 动和静之中都知道本源是什么 |
| 真常须应物 | 日常真心面对万事万物 |
| 应物要不迷 | 应对万事但不要沉迷其中 |
| 不迷性自住 | 不沉迷，本性自然安住 |
| 性住气自回 | 本性安住了，气就自动回来了 |
| 气回丹自结 | 气回来了，丹就自然结成了 |
| 阴阳生反复 | 阴阳反复交融 |
| 普化一声雷 | 气通经脉时如一声春雷 |
| 白云朝顶上 | 气沿督脉上升头顶 |
| 甘露洒须弥 | 津液（甘露）满口咽下 |
| 自饮长生酒 | 咽下自己的津液就是长生酒 |
| 逍遥谁得知 | 此中的逍遥，不修炼的人体会不到 |

---

### 4.2 入药镜（节选）

> 崔希范所作，内丹火候口诀。

```
先天气，后天气，得之者，常似醉。
日有合，月有合，穷戊己，定庚甲。
上鹊桥，下鹊桥，天应星，地应潮。
起巽风，运坤火，入黄房，成至宝。

水怕干，火怕寒，差毫发，不成丹。
铅龙升，汞虎降，驱二物，勿纵放。
产在坤，种在乾，但至诚，法自然。

盗天地，夺造化，攒五行，会八卦。
水真水，火真火，水火交，永不老。
水能流，火能焰，在身中，自可验。

是性命，非神气，水乡铅，只一味。
归根窍，复命关，贯尾闾，通泥丸。
```

**关键句子解读**：

| 原文 | 解读 |
|------|------|
| **先天气，后天气，得之者，常似醉** | 先天和后天之气合一后，人会进入一种微醺的舒适状态——这就是"气足"的标志 |
| **水怕干，火怕寒，差毫发，不成丹** | 水火既济的平衡极为精妙，稍有偏差（火候过头或不足）丹就结不成 |
| **盗天地，夺造化** | 修炼的本质是"逆取"天地的精华来补自身（即"顺则凡，逆则仙"） |
| **但至诚，法自然** | 再多的技巧，最终要归到"诚"和"自然"二字 |

### 4.4 悟真篇口诀（节选）

> 张伯端《悟真篇》，内丹经典。

**关于筑基**：
```
咽津纳气是人行，有药方能造化生。
鼎内若无真种子，犹将水火煮空铛。
```
→ 光练呼吸咽津只是表面功夫，丹田中若有"真种子"（精气充足）才是关键。否则就像用空锅烧水——白费功夫。

**关于炼精化气**：
```
坎电烹轰金水方，火发昆仑阴与阳。
二物若还和合了，自然丹熟遍身香。
```

**关于火候**：
```
纵识朱砂与黑铅，不知火候也如闲。
大都全藉修持力，毫发差殊不结丹。
```
→ 光知道材料（精气神）还不够，不懂火候（修炼的节奏和分寸）也没用。差一点点丹都结不成。

## 五、日常修持口诀

### 5.1 起床口诀

```
旭日东升，吾亦同醒。
东方青气，灌入我顶。
三呼三吸，浊气排尽。
今日修行，常清常静。
```

### 5.2 出门口诀

```
吾今出门，正气随身。
```

## 六、辟邪辟秽口诀

### 6.1 遇到不干净的地方

> 经过气场不好的地方（医院、殡仪馆、荒郊野外等）后可默念：

```
太上台星，应变无停。
驱邪缚魅，保命护身。
急急如律令。
```

同时配合动作：右手握固（四指包拇指成拳），藏于袖中或背后。

### 6.2 噩梦惊醒后

```
赫赫阳阳，日出东方。
断绝噩梦，辟除不祥。
吾奉太上老君，急急如律令。
```

同时：搓热双手，干洗脸 9 次，拍打全身。

默念金光咒最后四句：

```
内有霹雳，雷神隐名。
洞慧交彻，五炁腾腾。
金光速现，覆护真人。
```

## 七、特定场景口诀

### 7.1 考试 / 重要场合前的定心诀

```
心若冰清，天塌不惊。
万变犹定，神怡气静。
忘我守一，六根大定。
戒点养气，无私无为。
```

——出自《冰心诀》（虽是武侠小说中的口诀，但气韵暗合道家心法）

### 7.2 动怒时瞬间降火诀

```
呼——（缓缓呼出，如吹气）
心火下降，肾水上升。
水火既济，归入丹田。
```

配合动作：双手从胸前向下按，意想心火随手下行至丹田。

### 7.3 睡功调息诀

> 陈抟老祖睡功法：

```
先睡心，后睡眼。
闭目内观，耳不外听。
呼吸绵绵，意守丹田。
不知不觉，已入梦乡。
```

## 八、使用指南

### 8.1 哪些口诀适合日常用

| 口诀 | 使用频率 | 场景 |
|------|----------|------|
| 净心神咒 | 每次打坐前 | 日常修炼必备 |
| 金光神咒 | 每日一次 | 护身结界 |
| 观火法 | 每日一次 | 丹田聚气 |
| 五神存思 | 每周 2-3 次 | 深度修炼 |
| 入药镜 | 诵读即可 | 理解原理 |

### 8.2 口诀的使用原则

1. **诚心为要**：念诵时心存敬意，不在声音大小
2. **理解优先**：先看懂意思再诵，效果远好于有口无心
3. **选择适合的**：不必全学全用，选最有感应的几个深入
4. **默念为主**：日常以心中默念为主，不出声
5. **配合呼吸**：长句一口气诵完，自然停顿处换气

### 8.3 诵咒的音调

不需要像唱诵那样抑扬顿挫。平稳、匀速、一字一音即可。节奏如同缓缓的流水，不快不慢。
