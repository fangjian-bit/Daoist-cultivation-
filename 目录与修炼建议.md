# 目录与修炼建议

---

## 前言

本仓库收录了一套循序渐进的道家修炼资料，从入门筑基到进阶功法，从原理口诀到损耗恢复，供自学者按图索骥、稳步前行。

**使用原则**：先读原理，再练功法；先筑基，再进阶；遇损先恢复，切忌急功近利。

## 三、分阶段修炼建议

### 第一阶段：筑基期（第 1–3 个月）

**目标**：调身、调息、调心，建立每日固定练习习惯。

| 项目 | 建议 |
|------|------|
| 每日时长 | 早晚各 15 分钟起，稳定后可增至 40–60 分钟 |
| 核心功法 | 站桩 + 腹式呼吸 + 静坐 + 八段锦（先练三段） |
| 必读文档 | 初学者修炼手册、初学者功法详解 |
| 选读文档 | 原理与口诀（遇到疑问时查阅对应章节） |
| 作息 | 尽量 23:00 前入睡，卯时（5:00–7:00）为最佳练功时段 |

**筑基完成的自测标准**（详见进阶指南）：

- 站桩 30 分钟不觉枯燥，身体松静稳定
- 腹式呼吸自然深长，不需刻意控制
- 静坐 20 分钟杂念明显减少
- 睡眠质量改善，白天精神充沛
- 情绪较以前平和，不易大喜大怒

> 三项以上达标，且持续稳定两周以上，可考虑进入进阶准备。

---

### 第二阶段：深化筑基（第 4–12 个月）

**目标**：延长练功时间，引入逆腹式呼吸与浑圆桩，完整学习八段锦。

| 项目 | 建议 |
|------|------|
| 每日时长 | 60–90 分钟（可分早晚两次） |
| 核心功法 | 浑圆桩 + 逆腹式呼吸 + 完整八段锦 + 静坐 |
| 必读文档 | 道家修炼进阶指南、进阶功法详解 |
| 饮食作息 | 按进阶指南调整，七分饱、忌生冷、护子时 |

**此阶段切忌**：急于通小周天、过度追求气感、自行加大意念强度。

---

### 第三阶段：进阶期（1 年以上，视个人进度）

**目标**：在筑基扎实的前提下，学习意通小周天，动静结合。

| 项目 | 建议 |
|------|------|
| 核心功法 | 小周天（意通）+ 太极拳入门 + 五禽戏（选练） |
| 必读文档 | 进阶指南第三、九章；进阶功法详解小周天部分 |
| 口诀使用 | 按需查阅专业口诀汇编，日常修持类口诀优先 |
| 重要前提 | **小周天建议在明师指导下进行**，不可凭感觉硬通 |

## 四、每日修炼建议（精简版）

### 初学者每日方案（约 45 分钟）

| 时段 | 内容 | 时长 |
|------|------|------|
| 卯时 5:30–6:30 | 站桩 → 八段锦（三段）→ 静坐 | 40–60 分钟 |
| 亥时 21:30–22:00 | 腹式呼吸 → 简短静坐 → **收功** | 20–30 分钟 |

时间不够时的 **20 分钟精简版**：站桩 10 分钟 + 腹式呼吸 5 分钟 + 收功 5 分钟。

### 每次修炼必做三件事

1. **练前**：排空大小便，环境安静，衣着宽松
2. **练中**：松而不懈，意轻则灵，不追气感、不憋气
3. **练后**：按手册收功法操作，缓慢活动，避免立刻吹风或冲凉

## 五、通用修炼原则

1. **持之以恒优于一次练久**——每天 15 分钟，胜过周末练 3 小时
2. **身正心安**——姿势不正、心浮气躁时，宁可缩短时间或暂停
3. **损则先补**——熬夜、大怒、纵欲、过劳后，先读损耗恢复篇，勿强行练功
4. **不求神通**——修炼目标是身心康宁、性命双修，不是追求异象
5. **有疑则停**——出现持续头痛、心悸、失眠加重等异常，停止进阶功法，退回筑基或就医
6. **经典相辅**——手册推荐《道德经》《清静经》等，与练功并行，养德性
