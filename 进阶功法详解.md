# 进阶功法详解

---

## 前言

本文档是《修炼进阶指南》中功法的逐式细化拆解。

**阅读前提**：已完成至少三个月筑基，身心状态稳定，且已阅读过《修炼进阶指南》，理解进阶功法的整体框架。

> **核心安全声明**：进阶功法（尤其是小周天）若无明师指导，宁可只读不练。意念在体内的引导，**轻了无功，重了有害**。本文档的作用是以信息辅助理解，而非替代师承。

---

## 功法一：浑圆桩

### 概述

- **地位**：养生桩的升级版，从"局部聚气"到"周身一气"
- **前置条件**：养生桩能轻松站 30 分钟以上
- **时间**：卯时（5:00-7:00）最佳
- **时长**：20-45 分钟
- **口诀**：**"撑三抱七，似坐非坐。周身一家，内外合一。"**

### 与养生桩的区别

| 项目 | 养生桩 | 浑圆桩 |
|------|--------|--------|
| 手位高度 | 腹前（肚脐） | 胸前（两乳之间） |
| 手间距 | 两指尖相对 | 两掌与肩同宽 |
| 意念 | 守丹田 | 守全身 |
| 气感 | 丹田局部温热 | 全身气机贯通 |
| 体力消耗 | 低 | 中高 |
| 攻防含义 | 无 | 有（抱撑皆备） |

### 动作详解

**第一步：下盘**

与养生桩完全相同：
- 两脚平行与肩同宽
- 膝盖微屈不过脚尖
- 重心落于脚心

**第二步：身架**

1. 脊柱中正，虚领顶劲
2. 含胸拔背（与养生桩同）
3. 松腰敛臀，命门微后靠

**第三步：手臂（核心区别）**

1. 双手从体侧缓缓抬起，抬至胸前
2. 高度：两手与两乳同高，肘略低于肩
3. 宽度：两手掌心向内，间距与肩同宽（约一个篮球的距离）
4. 形状：如环抱一棵大树
   - "撑三"：手臂有微微向外撑开的意向（三成力）
   - "抱七"：手臂有环抱回来的意向（七成力）
5. 腋下含空，可容纳一个鸡蛋
6. 手指自然分开，不并拢也不过度张开
7. 手腕放松不屈

> **检验方法**：想象有人从侧面推你的手臂——你既有外撑的力不让其推入，又有回抱的力不使其拉开。这个"矛盾力"就是浑圆桩的精髓。

**第四步：意念（五层递进）**

| 层次 | 意念内容 | 适用阶段 |
|------|----------|----------|
| 第一层 | 抱树感应 | 想象胸前环抱一棵温暖的大树，双手和胸都能感到树传来的温热 |
| 第二层 | 水浴感应 | 想象身体泡在温水中，水有浮力托着双臂，周身都温暖 |
| 第三层 | 气球感应 | 想象腋下、两臂之间、两腿之间各夹一个气球，太松就飞了，太紧就破了 |
| 第四层 | 整体感应 | 不守一处，而守"全身"。感觉身体是一个整体的气团 |
| 第五层 | 空明感应 | 忘了身体，忘了气感，就是一"站"而已 |

> 每天站桩时，从前一层进入后一层，不跳级。

### 浑圆桩的"八面劲"

这是浑圆桩独有的体验——在静态中感受到八个方向的力：

| 方向 | 力的感觉 |
|------|----------|
| 上 | 头顶有上提之意（但不是耸肩） |
| 下 | 脚下生根，有下沉之意 |
| 前 | 胸有微含前拥之意 |
| 后 | 背有微靠后贴之意 |
| 左 | 左臂有外撑之意 |
| 右 | 右臂有外撑之意 |
| 内 | 四肢有向核心收拢之意 |
| 外 | 周身有向外膨胀之意 |

> 不是肌肉用力，是"意念在八个方向有轻微的张力感"。初学在第二三个月会渐渐体会到。

### 浑圆桩常见问题

| 问题 | 解决 |
|------|------|
| 手臂酸胀抬不住 | 降低手位（先抬到腹部高度，慢慢升高），缩短时间 |
| 肩膀僵硬 | 刻意放松肩膀，想象肩胛骨向下"滑" |
| 上重下轻（头重脚轻） | 更多关注脚下的感觉，意念沉在下盘 |

## 功法二：小周天功（意通周天）

### 概述

- **地位**：道家修炼的"第一道门槛"
- **前置条件**：丹田气感明显 + 会阴偶有跳动 + 逆腹式呼吸熟练
- **口诀**：**"吸升呼降，一息一周。勿忘勿助，气自行走。"**

### 小周天路线图

```
          百会（头顶）
            │
    玉枕关←┘
      （后脑）
        │
     夹脊关
  （肩胛骨间）
        │
      命门
    （后腰）
        │
     尾闾关
    （尾骨）
        │
     会阴（起点）
        │
     丹田 ←──────────┐
        │              │
       └─→ 会阴      │
             │         │
        督脉上升      │ 任脉下降
         （吸气）      │ （呼气）
             │         │
          百会 → 面部 ┤
                  │    │
               鹊桥 ──┘
              （舌抵上颚）
```

### 第一阶段：丹田聚气

**每次周天运行前必修步骤**：

1. 盘坐，全身放松，舌抵上颚
2. 逆腹式呼吸 3-5 分钟
3. 将全部意念轻轻放在丹田
4. 等待丹田出现温热感或胀满感
5. 不要急，有时候需要 5-10 分钟气才聚起来

> **判定标准**：丹田处有明显温热，好像里面有个小火炉。如果没有，不要强行运行周天——继续聚气即可。

### 第二阶段：气下会阴

丹田气足后，引导它下行：

**方法一（自然法，最安全）**：
- 继续意守丹田，不加引导
- 当丹田气足够充盈时，气会自然下行至会阴
- 你会感到会阴处微热、跳动或有气过之感

**方法二（意念轻引法）**：
- 吸气时，意念从丹田轻轻"看"向会阴
- 不是推动，只是转移观察点
- 如同手电筒的光从丹田移到了会阴

### 第三阶段：过尾闾关

尾闾关是周天第一关，位于尾骨尖端。

**过法**：
1. 确认会阴处有气感
2. 吸气时，意念从会阴轻轻移向尾骨尖端
3. 意念极轻，如一根羽毛扫过
4. 意念停在尾闾处
5. 下一次吸气时，感受尾闾是否有温热感
6. 有感觉了就说明过了

**若不通**：
- 轻轻提肛（收缩肛门的动作）3-5 次
- 想象尾闾如同一道门，轻轻推开
- 不要用力用意念"撞"——这是大忌

### 第四阶段：过命门

过了尾闾，气沿脊柱上行至后腰命门穴（肚脐正对的后方）。

1. 吸气时，意念从尾闾沿脊柱上行至命门
2. 在命门处微停
3. 体会后腰处发热——这是过命门的标志
4. 若长时间只有尾闾热而命门不热 → 守命门片刻，不急上行

### 第五阶段：过夹脊关

夹脊关位于两肩胛骨之间，是最窄的关卡。

1. 吸气时，意念从命门沿脊柱上行至夹脊
2. 夹脊处可能有酸胀感——正常
3. 若堵在此处，可轻轻转肩、扩胸 1-2 下
4. 呼气时意念停在夹脊，不往下走

**过夹脊的精要**：夹脊狭窄，意念要像一根丝线一样细，才能通过。

### 第六阶段：过玉枕关

这是周天最后一道关卡，也是最常卡住的一道。

1. 吸气时，意念从夹脊上行至玉枕（后脑枕骨处）
2. 玉枕关的标志：后脑发胀、发麻、发热
3. **若堵在此处**：
   - 轻轻收下巴，头微微后仰（调整枕骨角度）
   - 意念极轻，仿佛气是一缕烟，自然上升
   - 若仍堵，不要强通——继续守夹脊，等气足了自会过去

> **玉枕关特别提醒**：这是最容易出偏的一关。如果强行用意念推气过玉枕导致头胀头痛，立刻停止周天运行，回到守丹田。

### 第七阶段：到达百会

过了玉枕，气自然到达头顶百会穴：

1. 百会处有发麻、发胀、清凉或温热感
2. 在百会处微停一次呼吸
3. 不要长时间意守百会（防止气冲头）

### 第八阶段：任脉下降

1. **呼气**时，意念从百会向前下方移动
2. 经过眉心（上丹田） → 鼻梁 → 舌抵上颚处（鹊桥）
3. 舌尖会感到微微发麻——说明气过了鹊桥
4. 继续沿任脉下行 → 咽喉 → 膻中（胸口）→ 丹田
5. 到达丹田后，微停三次呼吸
6. 一圈完成

### 完整的呼吸配合

```
第一次吸气：会阴 → 尾闾 → 命门 → 夹脊 → 玉枕 → 百会
（在百会微停）
第一次呼气：百会 → 面部 → 鹊桥 → 咽喉 → 膻中 → 丹田
（在丹田停 1-3 次呼吸）
```

**每日运行圈数**：
- 初学（第一个月）：每次 3 圈
- 一个月后：每次 6 圈
- 三个月后：每次 9 圈
- 不要超过 9 圈（用意念运行的圈数越多，出偏风险越大）

> **金规则**：运行几次周天后，最后一定要意守丹田 3-5 分钟让气归位。不可运行完后就直接起身收功。

### 小周天修炼的"三不练"

| 情况 | 原因 |
|------|------|
| **疲劳不练** | 气虚时运行周天，容易上气不接下气 |
| **情绪波动不练** | 大怒、大悲、大喜后气机紊乱，不可周天 |
| **饱食不练** | 胃中有食，气聚于胃，无力运行周天 |

## 功法三：太极拳（简化 24 式入门）

### 概述

- **地位**：道家动功的最高形式之一
- **前置条件**：八段锦全套能熟练完成
- **频次**：每日清晨或傍晚，每次 20-40 分钟
- **口诀**：**"一动无有不动，一静无有不静。"**

### 学习路线图

| 周次 | 学习内容 | 说明 |
|------|----------|------|
| 第 1-2 周 | 起势 + 左右野马分鬃 + 白鹤亮翅 | 前三式打基础 |
| 第 3-4 周 | 左右搂膝拗步 + 手挥琵琶 + 倒卷肱 | 手脚配合 |
| 第 5-6 周 | 左揽雀尾 + 右揽雀尾 | 核心招式，棚捋挤按 |
| 第 7-8 周 | 单鞭 + 云手 + 单鞭 | 转身与手法 |
| 第 9-10 周 | 高探马到闪通臂 | 中间段 |
| 第 11-12 周 | 转身搬拦捶到收势 | 结尾段 |
| 第 13 周起 | 全套连贯练习 | 追求连绵不断 |

### 起势详解

1. 双脚并拢，自然站立
2. 重心移到右腿，左脚分开与肩同宽
3. 双手缓缓向前上抬起，与肩同高
4. 屈膝下沉，双手缓缓下按至腹前
5. 整个过程配合呼吸：起手时吸气，下按时呼气

### 云手详解（核心招式）

云手是太极拳中最体现"圆转"的招式：

**左云手**：
1. 重心在右腿，腰向右转
2. 右手在体前画圆（向右→向上→向左）
3. 腰向左转，重心过渡到左腿
4. 左手在体前画圆

**练习要点**：
- 腰为主宰，手只是跟随腰部转动
- 如同在搅动一缸水——手不是自己动，是被腰"带"动的
- 上下配合：手不过眉，手不低于腹

### 太极拳的"十要"

| 序号 | 要领 | 详解 |
|------|------|------|
| 1 | 虚领顶劲 | 头顶如线悬 |
| 2 | 含胸拔背 | 胸微含，背舒展 |
| 3 | 松腰 | 腰部松沉 |
| 4 | 分虚实 | 两腿重心分明 |
| 5 | 沉肩坠肘 | 肩松肘下沉 |
| 6 | 用意不用力 | 意念领形，不使蛮力 |
| 7 | 上下相随 | 手到脚到，周身协调 |
| 8 | 内外相合 | 外形与内气统一 |
| 9 | 相连不断 | 动作如流水不间断 |
| 10 | 动中求静 | 运动中保持内心安静 |

### 太极拳呼吸配合

初学时不强求呼吸配合，动作熟练后再加入：

| 动作方向 | 呼吸 |
|----------|------|
| 起手、开、发、升 | 吸气 |
| 落手、合、收、降 | 呼气 |

**引导意念**：始终把三分意念放在丹田，七分留意动作。丹田如陀螺的重心点，任身体如何旋转，重心始终在丹田。

## 功法四：五禽戏

### 概述

- **地位**：华佗所创，模仿虎鹿熊猿鸟五种动物
- **时间**：每周 2-3 次，与八段锦或太极拳交替练习
- **时长**：全套约 20 分钟
- **口诀**：**"形神兼备，以意领形。"**

### 虎戏——练骨

**虎举**：
1. 双手成虎爪（五指弯曲如钩），从体侧缓缓上举
2. 举过头顶后翻掌，爪心向上
3. 缓缓下落还原
4. 做 3-5 次

**虎扑**：
1. 下蹲，双手成虎爪按于膝旁
2. 全身蓄劲后，向前扑出
3. 如猛虎扑食，双手前伸，身体前探
4. 收回，重复

**功效**：强筋健骨，增强肺活量。

**意念**：想象自己是深山中的猛虎，目光有威，力量由脊背发出。

---

### 鹿戏——练筋

**鹿抵**：
1. 一手成鹿角状（中指无名指弯曲，其余三指直立），向对侧转腰伸出
2. 头随着手的方向转动
3. 左右交替

**鹿奔**：
1. 弓步向前，一手前探如鹿角
2. 后腿蹬地，向前奔出
3. 收腿换步

**功效**：舒展筋脉，强腰健肾。

**意念**：鹿回头时的回望，带着警觉与灵动。

---

### 熊戏——练肉

**熊运**：
1. 两脚平行，膝微屈
2. 两手成熊掌（掌心向下，五指微曲），在腹前画圆
3. 肚皮随着手动而起伏
4. 左右交替画圈

**熊晃**：
1. 左右摇晃上身，如熊之笨重
2. 步法沉重，一步一蹲

**功效**：健脾胃，消化好，肉实不虚。

**意念**：想象自己是冬眠初醒的熊，动作缓慢、沉重、带着浑厚的内劲。

---

### 猿戏——练神

**猿提**：
1. 双手成猿爪（撮拢五指尖），快速上提至胸前
2. 同时缩脖、耸肩
3. 头左右转动张望

**猿摘**：
1. 如猿猴攀枝摘果，一手勾住（树枝），一手探出（摘果）
2. 身形灵动，动作敏捷

**功效**：养心神，增强肢体灵活度。

**意念**：想象白猿摘桃，眼神灵动，动作轻捷。

---

### 鸟戏——练皮

**鸟伸**：
1. 双手在体侧平举，如大鹏展翅
2. 缓缓上下扇动
3. 配合呼吸

**鸟飞**：
1. 单腿站立，另一腿后伸
2. 两臂如翅膀展开
3. 保持平衡

**功效**：舒肺气，活关节，调节平衡感。

**意念**：想象自己是一只在云中滑翔的仙鹤，轻灵、自在、呼吸深长。

## 四种进阶功法的组合方案

### 方案一：大小周天为主（约 60 分钟）

适合以静功为主攻方向的修炼者。

| 顺序 | 功法 | 时长 |
|------|------|------|
| 1 | 浑圆桩 | 15 分钟 |
| 2 | 逆腹式呼吸 | 5 分钟 |
| 3 | 丹田聚气 | 5-10 分钟 |
| 4 | 小周天运行 | 9 圈约 15 分钟 |
| 5 | 意守丹田收功 | 5-10 分钟 |

### 方案二：动静结合（约 70 分钟）

适合希望全面发展的修炼者。

| 顺序 | 功法 | 时长 |
|------|------|------|
| 1 | 浑圆桩 | 15 分钟 |
| 2 | 八段锦或太极拳 | 20 分钟 |
| 3 | 静坐（守丹田或意通周天） | 25 分钟 |
| 4 | 收功 | 10 分钟 |

### 方案三：动功日 + 静功日交替

| 日期 | 类型 | 内容 |
|------|------|------|
| 周一 | 静功日 | 浑圆桩 + 小周天 |
| 周二 | 动功日 | 太极拳 + 五禽戏 + 短时静坐 |
| 周三 | 静功日 | 浑圆桩 + 小周天 |
| 周四 | 动功日 | 八段锦 + 太极拳 + 短时静坐 |
| 周五 | 静功日 | 浑圆桩 + 小周天 |
| 周六 | 综合日 | 全套功法，时间可适当延长 |
| 周日 | 休息日 | 仅做简单的腹式呼吸 10 分钟 |
